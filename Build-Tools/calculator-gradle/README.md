# Lab 1 - Calculator Gradle

## Objective

Build and package a Java calculator application using Gradle.

## Tasks

- Install Java 17
- Install Gradle
- Clone the project
- Run unit tests
- Build the application
- Generate the JAR file
- Run the application

## Commands

```bash
gradle test
gradle build
java -jar build/libs/calculator.jar
```
